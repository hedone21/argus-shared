# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities privately. Do **not** open a public
issue for security-sensitive reports.

- Use [GitHub Security Advisories](https://github.com/hedone21/argus-shared/security/advisories/new)
  to report privately, or
- email the maintainer at `hedone21@gmail.com`.

We aim to acknowledge reports within a few days and will coordinate a fix and
disclosure timeline with you.

## Supported Versions

This project is pre-1.0. Security fixes are applied to the latest released
version on a best-effort basis.
